# 밤하늘 광공해와 별 관측 가능성 분석

**분석 목적** : 도시 조명으로 인한 광공해가 천체 관측 가능성에 미치는 영향을 정량적으로 분석한다. 인공조명 밀집 지역(도시)과 천체 관측지의 별 밝기 손실률을 비교한다.

**분석데이터** : NASA Earth at Night, Light Pollution Map, 인구밀도 데이터

**분석** : 시계열 분석(조도 변화에 따른 야간 하늘 밝기 추세 확인), 로지스틱 회귀(인공조도(밝기 단위) → 별 관측 가능 지역(1/0) 예측)

**인사이트** : 도시 지역은 광공해가 심해 별 가시성이 낮을 것으로 예상

**결론(예시)** : 광공해 지수가 높을수록 별 관측 확률이 현저히 감소, 인구밀도가 높은 지역일수록 관측 불가 구역 비율이 높음

**서비스 아이디어** : 광공해 수준을 기반으로 별 보기 좋은 곳을 추천해주는 지도형 플랫폼
